# **App Name**: Ping Pong Champions

## Core Features:

- User Authentication: Register and log in via email to manage profiles and league participation.
- League Management: Administrators can create, edit (name, description), and delete leagues.
- League Membership: Users can join existing leagues as players or become administrators of new leagues.
- Match Result Entry: Players enter ping pong match results (1:0 or 2:0 or 2:1 or 3:0 or 3:1 or 3:2).
- Result Moderation: Administrators can edit or delete match results to maintain data accuracy.
- ELO Ranking: Calculate and display league rankings based on players' ELO points, updated after each match.
- Player Statistics: Provide simple statistics for each player, including head-to-head records and ELO point/ranking development graphs.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trustworthiness and competence.
- Background color: Light gray (#F0F2F5), a subtle variation of the primary color, to ensure readability and minimize distractions.
- Accent color: Vibrant orange (#FF9800), an analogous hue, to draw attention to important interactive elements.
- Body and headline font: 'PT Sans', a sans-serif typeface with a modern and slightly warm look, is used for both body and headline texts.
- Mobile-first responsive design ensuring a clear and easy-to-use interface on all devices.
- Simple, geometric icons to represent actions and data points.
- Subtle transitions and animations to enhance user experience without being distracting.
